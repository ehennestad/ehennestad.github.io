Nansen

NANSEN (Neuroscience ANalysis Software ENsemble) is a package with modules for building a complete processing pipeline and exploring data using interactive graphical user interfaces.

Currently available modules include existing toolboxes widely used by the neuroscience community, such as NoRMCorre, CaImAn and Suite2P


Session Browser

Interactive table with sorting and filtering capabilities

Create metadata collections for specialiced analysis pipelines

Run prepackaged methods in batch

Imviewer

Open and explore image arrays or virtual image stacks.

Open and configure motion correction tools like NoRMCorre or FlowRegistration

Manually annotate rois or run autosegmentation algorithms like CNMF, Suite2P or EXTRACT

Signal Viewer

Interactively view and process signals

Dashboards

Fovmanager

Keep track of fov locations, injections spots in real world stereotaxic coordinates

Visualize FoV and RoI statistics directly on map
